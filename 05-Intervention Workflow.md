# Intervention Workflow

## 1. Soft Alert Flow
- Alert appears  
- Homeroom teacher notified  
- Parent contacted  
- Notes logged

## 2. Critical Alert Flow
- Alert appears  
- Home visit recommended  
- Principal informed  
- Support action planned

## 3. Parent Interaction
- Message sent  
- Parent reply logged

## 4. Outcome Logging
- Problem resolved  
- Ongoing  
- Escalated
